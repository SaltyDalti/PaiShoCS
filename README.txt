
This folder contains all Pai Sho C# scripts generated for your custom Unity rules engine.

To use:
1. Unzip this folder inside your Unity project root.
2. Move the contents into your Assets/Scripts/ structure if needed.
3. Commit to your GitHub repo at https://github.com/SaltyDalti/PaiShoCS

Modules included:
- GameManager
- TileSelector
- ReserveManager
- CaptureManager (with radiant disharmony)
- PlacementValidator (with port-blocking prevention)
- TileLifecycleManager (wilting system)
- HarmonyManager
- PieceMovement

Each file is ready for direct integration.
